Admin Service - Moderation Endpoints

All endpoints require Bearer token with role=admin.
  • GET /admin/users - list users with status/date filters
  • GET /admin/users/{userId} - user details
  • PATCH /admin/users/{userId} - set active|deactivated|restricted
  • GET /admin/posts - list posts with pagination filters
  • GET /admin/posts/{postId} - post + author details
  • DELETE /admin/posts/{postId} - soft delete + Redis timeline cleanup
  • GET /admin/stats - aggregate totals
